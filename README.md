# Community Connect - Complaint Management System

## Project Overview
Community Connect is a web-based complaint management system designed for apartment residents to register and track complaints efficiently.

## Features
- Complaint registration (Water, Electricity, Maintenance, Security)
- Supabase database storage
- Email notification system
- Admin complaint tracking
- Responsive UI

## Technologies Used
- React (Vite)
- TypeScript
- Tailwind CSS
- Supabase
- Flask (Backend)
- SMTP Email Service

## Installation

1. Clone the repository:
   git clone <your-repository-url>

2. Install dependencies:
   npm install

3. Start the development server:
   npm run dev
